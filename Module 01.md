### 1. What is Artificial Intelligence?

Artificial Intelligence (AI) refers to computer systems that perform tasks we would normally associate with human thinking — recognising speech, understanding text, making predictions, or classifying images.

The term was coined in 1956 at Dartmouth College, but the ideas go back further. Early AI systems were **rule-based**: programmers wrote explicit instructions for every situation. These worked well for narrow problems but broke down quickly in the complexity of the real world.

**Machine Learning (ML)** changed the paradigm. Instead of writing rules, we feed the machine *examples* — thousands or millions of labelled data points — and let it discover the patterns itself.

**Deep Learning** is a subset of ML that uses layered neural networks inspired (loosely) by the human brain. It powers most of what we recognise as modern AI: image recognition, language translation, voice assistants.

---

### 2. Three Types of Machine Learning

| Type | How it learns | Example |
|------|--------------|---------|
| **Supervised** | From labelled examples (input → correct answer) | Spam filter, disease diagnosis |
| **Unsupervised** | Finds patterns in unlabelled data | Customer segmentation, anomaly detection |
| **Reinforcement** | Trial and error with rewards/penalties | Game-playing agents, robot navigation |

Most practical AI you will encounter is **supervised learning**.

---

### 3. AI in Africa Today

AI is not a future technology for Africa — it is already here:

- **Mobile money fraud detection** (M-Pesa, MTN Mobile Money) uses ML to flag suspicious transactions in real time
- **Agricultural advisory apps** (e.g. Zindi challenges, Farmcrowdy) use satellite imagery + ML to advise smallholder farmers
- **Health diagnostics** — AI tools for malaria detection from blood smear images are being piloted in several SSA countries
- **Language technology** — projects like Masakhane are building NLP models for African languages
- **Content moderation** — social media platforms deploy AI moderation that often performs poorly on African languages and contexts

**Discussion question:** Can you name an app or service you use that is likely powered by AI?

### 4. Key Vocabulary

| Term | Definition |
|------|-----------|
| Algorithm | A step-by-step set of instructions for solving a problem |
| Training data | The dataset used to teach a machine learning model |
| Model | The output of training — a mathematical function that makes predictions |
| Feature | An input variable used by the model (e.g. age, region, word frequency) |
| Label | The correct answer the model is trying to predict |
| Accuracy | The percentage of predictions the model gets right |
| Bias | Systematic errors in model output, often reflecting inequities in training data |

---

### 5. Critical Reflection

AI systems are built by people — and they reflect the choices, assumptions, and blind spots of their creators. Some questions worth asking about any AI system:

1. **Who built it, and for whom?**
2. **What data was it trained on — and whose lives does that data represent?**
3. **Who is harmed if it gets it wrong?**
4. **Who is accountable when it fails?**

These are not just philosophical questions. They have direct consequences for African citizens interacting with AI systems designed elsewhere, for different populations, with different assumptions.
