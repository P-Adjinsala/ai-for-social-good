# 🌍 AI for Social Good — Course Repository

> **A practical introduction to Artificial Intelligence and civic technology for university students in Africa**

**Instructor:** Pantouin Adjinsala  
**Affiliation:** University Lecturer & Civic Tech Contributor, AfricTivistes CitizenLab Cameroon  
**Level:** Undergraduate (Year 2–3) | No prior AI experience required  
**Language:** English (French translation in progress)  
**Format:** 4 modules × 2 weeks each | Lectures + hands-on notebooks + projects  

---

## 🎯 Course Objectives

By the end of this course, students will be able to:

- Explain what AI is, how it works, and where it is already present in African daily life
- Critically assess the social, ethical, and governance implications of AI systems
- Use Python and open data to perform basic data analysis relevant to civic problems
- Design and prototype a simple AI-assisted civic tech solution
- Apply an ethical framework to AI development and deployment decisions

## 🚀 Getting Started

All notebooks run on **Google Colab** — no local installation required.

1. Click any `notebook.ipynb` file in this repository
2. Click the **"Open in Colab"** badge at the top of the notebook
3. Run cells with `Shift + Enter`

Alternatively, clone the repo:
```bash
git clone https://github.com/YOUR_USERNAME/ai-for-social-good.git
cd ai-for-social-good
pip install -r requirements.txt
```

## 🌱 African Context

This course is designed with the African student experience in mind:

- **Examples draw from Cameroon, Nigeria, Kenya, Ghana, and the broader SSA region**
- **Datasets reflect real civic challenges** — health access, misinformation, electoral participation
- **Ethical discussions centre on African regulatory contexts**, data sovereignty, and community consent
- **Low-bandwidth friendly** — all notebooks run on free Colab tier, datasets are kept small
